# cats-bio

Чистый современный **сайт-био** на React + Vite + TypeScript + Tailwind.

Это переделанная и очищенная версия https://cats.tg/ на фреймворке.

## Особенности

- Полностью сохранён оригинальный дизайн и все пасхалки
- Написан на React (компоненты, TypeScript)
- Легко редактировать контакты в `src/App.tsx`
- Одинокий котик (Oneko) следует за курсором
- Звёздное небо, искры, typewriter, glitch, Konami и т.д.
- Yandex Music статус — опционально (graceful fallback)

## Запуск (локально с Яндекс Музыкой)

```bash
npm install
npm run dev:full
```

- Прокси для Яндекс Музыки запустится на порту 4000
- Фронтенд на http://localhost:3000

Смотри **SERVER.md** — там как получить токен и настроить `.env.server`

> **Примечание**: Токен хранится только на сервере (в .env.server), никогда не попадает в браузер.

## Сборка

```bash
npm run build
npm run preview
```

## Как кастомизировать

1. Контакты — массив `SOCIALS` в `src/App.tsx`

3. Цвета и стили — в `src/index.css` (или через CSS-переменные)
4. Добавить новые секции — просто в `App.tsx`

## Пасхалки (те же что и в оригинале)

- `?` — подсказка по клавишам
- `C` — смена акцентного цвета
- Двойной клик по фото — сердечки
- Konami code (↑ ↑ ↓ ↓ ← → ← → B A) — дождь из котиков
- Напиши `meow` — следы лапок
- Правый клик по фото — "no stealing"

Готово к использованию как личный био-сайт.
